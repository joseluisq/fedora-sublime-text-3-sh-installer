# Fedora / Sublime Text 3 Shell installer
> Sublime Text 3 shell installer for Fedora Linux x64

## Install

```sh
$ wget -qO- http://git.io/fed-st3-sh | sudo sh
```

## Usage

```sh
$ subl my-directory-or-file-path
```
